ABOUT
=====

This module allows to show a call button in mobile view.

Created by Aris Priyantoro
Site: http://aris.pituruh.com.

USAGE:
1. Activate the module
2. Go to: admin/config/user-interface/callnow to configure it
3. The call button will be show in public view when the screen size is less than or equal to 650px
